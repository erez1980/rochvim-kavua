# מדריך הרצה — סקרייפר GPX של "רוכבים קבוע"

הסקרייפר מוריד קבצי GPX למסלולים מ-7 מקורות, שומר אותם בתיקיית `gpx/`,
ומעדכן את `index.html` כך שכל מסלול שירד עבורו קובץ יקבל **כפתור הורדה מקומי**
(המשתמשים יורידו ישירות מהאתר שלך, בלי לצאת לאתר חיצוני).

## דרישות מקדימות

1. מותקן Python 3 (בדוק עם: `python3 --version`)
2. הספרייה requests. אם חסרה, התקן:
   ```
   pip3 install requests
   ```

## איך מריצים

1. ודא ש-3 הקבצים האלה נמצאים באותה תיקייה:
   - `gpx_scraper.py`  (הסקרייפר)
   - `tracks.json`     (רשימת 144 המסלולים)
   - `index.html`      (האתר)

2. פתח טרמינל בתיקייה והרץ:
   ```
   cd ~/Downloads/rochvim-site
   python3 gpx_scraper.py
   ```

3. הסקרייפר ירוץ מסלול-מסלול ויציג התקדמות חיה:
   - ✓ = נמצא והורד (עם שם המקור)
   - ✗ = לא נמצא באף מקור

4. בסיום תקבל דוח: כמה קבצים ירדו, פילוח לפי מקור, ורשימת מסלולים שלא נמצאו.

## מה הסקרייפר עושה

- **מקור 1 — קק"ל:** נכנס לדף המסלול ומחלץ את קישור ה-GPX הרשמי. הכי אמין.
- **מקורות 2-7 (fallback):** סריקת דף המקור, Israel Hiking Map, OSM Overpass,
  osmrm, TwoNav, ו-Wikiloc — מנסה אותם בזה אחר זה עד שאחד מצליח.
- שומר כל קובץ בתיקיית `gpx/` עם שם המסלול.
- מעדכן את `index.html` — מוסיף `localgpx` לכל מסלול שירד, והכפתור הופך להורדה מקומית.
- שומר `gpx_map.json` עם השיוכים (לתיעוד).

## אחרי ההרצה — העלאה ל-GitHub

```
git add index.html gpx/
git commit -m "GPX מקומי למסלולים"
git push
```

## הערות חשובות

- **חלק מהמסלולים לא יימצאו** — זה תקין. לא לכל מסלול יש GPX ציבורי נגיש.
  מסלולים שלא נמצאו ימשיכו להציג כפתור ניווט תקין (לא נשבר כלום).
- **Google Drive / sendspace** (שחלק מהמקורות מארחים בהם) חוסמים הורדה אוטומטית —
  אלה יידלגו. זה מגבלת המקורות, לא הסקרייפר.
- **הרצה חוזרת:** אפשר להריץ שוב בכל עת — קבצים שכבר ירדו יידרסו בגרסה עדכנית.
- הסקרייפר משהה שנייה בין בקשות מתוך כבוד לשרתים. ל-144 מסלולים זה ~5-7 דקות.
- אם תרצה להוסיף קובץ ידנית: שמור אותו ב-`gpx/` בשם המסלול, והוסף למסלול
  ב-index.html את השדה `localgpx:"gpx/שם_הקובץ.gpx"`.
