כלי פרטי. מעלה קבצי GPX/KML/KMZ ישירות ל-GitHub ומעדכן את המסלול. רק עבורך.

🔑 התחברות ל-GitHub

איך יוצרים Token? (פעם אחת)
  1. github.com ← Settings ← Developer settings ← Personal access tokens ← Fine-grained tokens
  2. Generate new token. תן שם (למשל "rochvim admin").
  3. Repository access ← Only select repositories ← rochvim-kavua
  4. Permissions ← Repository ← Contents ← Read and write
  5. Generate, העתק את ה-Token, הדבק כאן.

ה-Token נשמר רק בדפדפן הזה (localStorage), נשלח רק ל-GitHub, ולא נכנס לקוד.
